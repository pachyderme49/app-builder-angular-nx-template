# App

Projet Angular généré avec [Angular CLI](https://github.com/angular/angular-cli) (build Vite/esbuild, sans Nx).

## Serveur de développement

```bash
npm start
```

Puis ouvrir `http://localhost:4200/`.

## Build

```bash
npm run build
```

Les artefacts sont dans `dist/`.

## Commandes

- `npm start` — serveur de dev
- `npm run build` — build production
- `npm run watch` — build en mode watch (développement)
